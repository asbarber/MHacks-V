Detailed Description

Represents a Myo device with a specific MAC address.

This class can not be instantiated directly; instead, use Hub to get access to a Myo. There is only one Myo instance corresponding to each device; thus, if the addresses of two Myo instances compare equal, they refer to the same device.

Examples:
BackgroundService.java, HelloWorldActivity.java, MultipleMyosActivity.java, and MyoGlassService.java.
